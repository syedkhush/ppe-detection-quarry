# PPE Detection using YOLOv8

A Computer Vision model that detects Personal Protective Equipment (PPE) on industrial workers in real-time.

## Classes Detected
- Hard Hat
- Safety Vest  
- Gloves
- Mask
- Safety Boots
- Person

## Results
- mAP50: 95.1%
- Trained on 2,114 images

## Motivation
Built by Syed Sami — 3 years Operations Manager at a granite quarry where PPE compliance was a daily safety challenge.

## Tech Stack
- YOLOv8n (Ultralytics)
- Python
- Roboflow Dataset
